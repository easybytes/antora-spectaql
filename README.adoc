=== Installation

Install using npm

[source,shell script,subs="+attributes"]
----
npm install {package-name}
----

Add the extension to the the `antora-playbook.yml` playbook:

[source,yml,subs="+attributes"]
----
pipeline:
  extensions:
  - require: "{package-name}"
----
